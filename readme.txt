== Description ==

The Custom Forgot Password Mail plugin for WordPress is a free tool that lets you send personalized emails to users when they attempt to reset their passwords.<br /><br />
<strong>Manage your users efficiently</strong><br />

<strong>Customize your recovery emails:</strong>

<strong>Boost user engagement on your WordPress site by providing clear, easy-to-follow instructions in your recovery emails with the Custom Forgot Password Mail plugin!</strong>
Many users appreciate immediate support.<br />

The Custom Forgot Password Mail plugin is developed and maintained by <a href="https://www.galaxyweblinks.com/">Galaxy Weblinks</a> a technology company dedicated to delivering high-quality products at affordable prices on time.


== Installation ==

= Plugin Installation =
1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site,
2. Activate the plugin through the 'Plugins' menu in WordPress,
3. Visit the "Custom Forgot Password Mail" administration page to setup the plugin.
